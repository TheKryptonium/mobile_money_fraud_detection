# MOBILE MONEY FRAUD DETECTION

In the African socio-economic context, mobile money plays a crucial role, even more so than banks, in the financial transactions of most people. However, with the increasing use of this payment method over time, reported cases of fraud have also risen. It is with this in mind that we decided to develop a classification model to effectively detect fraud in mobile money transactions.
Technically, the goal is to reach an accuracy rate of at least 92% with classical ML techniques.
